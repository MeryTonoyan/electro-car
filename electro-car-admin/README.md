# ⚡ ELECTRO — Premium Electric Car Showroom

ELECTRO is a modern web platform for importing and selling premium electric vehicles. This project is built using a combination of **Next.js** and **Bootstrap**, ensuring ultra-fast performance, top-tier SEO optimization, and a user-friendly Admin Dashboard.

---

## 🚀 Key Features

* **Home Page (Hero Section):** An engaging landing area featuring interactive animations, counting statistics, and sleek floating cards.
* **Car Catalog:** A dynamic client-side filtering system organized by brand (Tesla, Porsche, BYD).
* **Custom Order System:** An interactive submission form allowing customers to request custom vehicle imports.
* **Admin Panel:** A dedicated, clean dashboard for administrators to view statistics, list vehicles, and add new cars to the inventory.
* **Responsive Design:** Flawlessly adapted to all screen sizes, including Mobile, Tablet, and Desktop.

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router), React, TypeScript
* **Styling:** Bootstrap 5, CSS3 (Custom variables, Keyframe animations)
* **Icons:** SVG (Inline optimized vector icons)
